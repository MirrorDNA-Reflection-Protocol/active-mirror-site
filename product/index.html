The v0.1 loop is intentionally narrow.

The first job is not to replace every app. It is to help one real workflow move from scattered context to a useful artifact with an auditable receipt.

  1. 01Bring the decision

    Start with a decision, problem, transcript, launch plan, or messy note.

  2. 02Reflect before predicting

    Active Mirror separates context, assumptions, risks, and blind spots before suggesting a next step.

  3. 03Create the artifact

    Turn the session into a summary, plan, memo, brief, checklist, or working board.

  4. 04Read the receipt

    See what was used, what was excluded, which route ran, and what memory was proposed.

  5. 05Approve the memory

    Continuity grows only through explicit approval, edit, temporary save, export, forget, or block.
