Privacy starts with less collection.

Last updated June 21, 2026. This page describes how the public site and workspace handle privacy.

Public site

This website does not need personal details to explain the product or let you try the workspace.

Browser mirror

The workspace can create a board in your browser without sending your input to an external service. Extra help should be disclosed before use.

Extra help

When external help is enabled, calls should happen through a protected server path. API keys must not be exposed in website code.

Memory

Private context should not become future context unless the user approves the memory decision. Receipts should show what was saved, ignored, or excluded.

Contact

Questions about privacy can be sent to hello@activemirror.ai.
