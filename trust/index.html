Trust gates

The browser workspace stays conservative: no exposed API keys, no silent memory, no unsupported claims promoted as fact.

No exposed keysAPI keys stay behind protected server paths, never in page code.
No silent memoryMemory promotion is a user-visible decision with receipt context.
Scoped model helpModels receive the approved task, not the whole private workspace by default.
Browser-only backupIf cloud help is unavailable, the browser can still create a structured workspace and record what happened.
Unsupported stays uncertainClaims without support must stay marked as uncertain, partial, inferred, or excluded.
No public client claimsInstitutional or client proof waits for explicit sign-off before appearing on the public site.
